# Admin Web

Owner: Mason, Admin Frontend Agent

Target stack: Next.js / TypeScript.

## Dependency-Free Preview

The preview server uses only Node built-ins:

```text
node scripts/dev-server.mjs
```

Default port: `4301`.

Preview routes:

- `/login`
- `/dashboard`
- `/users/review`
- `/users`
- `/reports`
- `/wallets`

Initial responsibility:

- Admin login shell
- User review queue
- Profile approval and rejection
- Report queue
- Suspension controls
- Basic metrics dashboard
