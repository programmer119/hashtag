# Hashdate Homepage

Owner: Lena, Homepage Frontend Agent

Lead integration: Codex

Static Next.js-style public homepage skeleton for the Hashdate MVP.

## Scope

- Landing page
- Privacy policy
- Terms of service
- Support
- Account deletion
- Download
- Safety

This folder declares a Next.js package/config skeleton, but dependencies have not been installed yet. The files are structured for a future Next.js app-router implementation under `app/`.

## Run Status

Local dev commands are declared in `package.json`, but they cannot run until dependencies are installed.
