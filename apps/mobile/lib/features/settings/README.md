# Settings Feature

Owns account, notification placeholder, safety/report paths, logout, and account deletion request entry.

First screens:

- `SettingsScreen`
- `AccountSettingsScreen`
- `SafetySettingsScreen`

State:

- `SettingsController`

API:

- `GET /me`
- `PATCH /me/settings`
- `POST /reports`
- `POST /me/deletion-request`
