- `DIAMOND_POLICY.md`
- `MODERATION_POLICY.md`

- `ACCEPTANCE_CRITERIA.md`

## Member Review

- New profiles require admin approval before discovery exposure.
- Rejected profiles should receive a short rejection reason selected from controlled admin options.
- Suspended users cannot swipe, chat, purchase, or appear in discovery.
- MVP review is profile-quality review, not income, document, or government identity verification.

## Diamond Rules

- Users have a diamond wallet.
- Premium profile detail unlock costs 10 diamonds in MVP.
- Repeated viewing of an already unlocked profile must not deduct again.
- All wallet changes must be stored as immutable ledger entries.
- Likes, matches, chat, reports, and blocks are free in MVP.

## Matching Rules

- A like creates a pending interest record.
- A mutual like creates a match.
- Chat opens only after a match.
- Blocked or suspended users cannot create new matches.

## Safety Rules

- Reports create admin review items.
- Admins can suspend or reinstate users.
- Chat and profile content should be kept auditable for abuse review.
- Users can block from profile detail or chat.
- Suspended users are removed from discovery eligibility immediately.
